Important FAQ

What is Document Object Model?

The Document Object Model (DOM) is a structure that represents a web page's content as a tree of objects, allowing JavaScript to interact with and modify the HTML, CSS, and content of the page.

How do you select an element from DOM?

To select an element from the DOM, I use JavaScript methods like:

document.getElementById()
document.querySelector()
document.querySelectorAll()

What is event delegation in the context of DOM and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element to handle events for its child elements. It's useful because it improves performance and makes managing events easier, especially when you have many child elements or dynamically added elements.

How do you manipulate an element's attributes and styles using DOM?

To manipulate an element's attributes using the DOM, we can use methods like:

element.setAttribute()
element.getAttribute()
element.removeAttribute()

To manipulate an element's styles:

element.style.property = 'value'
